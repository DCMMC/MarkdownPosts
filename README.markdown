# Notes 格式说明

记得在每一个Markdown文件开头添加一下模板：

```
---
title:  "博文标题"
date:   2019-01-16 10:42:06 +0800 # 时间
author: Bill Kevin # 作者
mathjax: true # 是否开启 mathjax
categories: Test # 分类
img: /images/demo0.jpg # 封面图片(暂时在 /images 是在 /source/_posts/images 中)
tags: # 标签(多个)
  - Markdown
---
```
